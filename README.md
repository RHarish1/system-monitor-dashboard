# System Monitoring Dashboard

A simple Flask-based web dashboard for real-time system monitoring using `psutil`. This dashboard displays important system metrics such as CPU usage, memory usage, and disk usage in a clean and simple web interface.

## Features
- Real-time monitoring of:
  - **CPU usage** (% of total capacity)
  - **Memory usage** (available and used memory)
  - **Disk usage** (total, used, and free disk space)
- Built using **Flask**, **Python**, **HTML**, and **CSS**.
- Simple and lightweight solution for monitoring basic system performance.
